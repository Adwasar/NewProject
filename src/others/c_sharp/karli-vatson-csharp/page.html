Аннотация

С# - это новый язык программирования Microsoft, предназначенный для ее новой платформы .Net Framework.

книга является незаменимым пособием для тех, кто учится создавать программы на C#. Ключевые понятия C# и .NET объясняются в ней последовательно, по мере увеличения знаний читателя. После детального знакомства с основами C# автор переходит к подробному рассмотрению объектно-ориентированного программирования на C#, а затем обращается к созданию Windows-приложений на C#. В заключение приводятся два практических примера, которые дают всеобъемлющее представление о приложениях на C# и о том, как они работают.
